The test data in this folder is a sub-set of the example data from the official
OpenGIS schema repository at:

http://schemas.opengis.net

A complete set of schemas is available for download at:
http://schemas.opengis.net/SCHEMAS_OPENGIS_NET.zip

This test data is used by the Unit Test cases of 'opengis-client-kotlin' to verify
error-free unmarshalling of OpenGIS data.
